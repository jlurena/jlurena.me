Solitaire Chess is a JavaFX Game (v2) built by hand (no GUI Builders).

Below you will find a live Java demo. It is self-signed without a certificate so to play you will have to add my website as a trusted site in Java's settings.

If you do not wish to allow this, you can view the source code on GitHub or download the executable jar by clicking here.

This project first started out as a very simple class project in the Fall of 2015. The following summer I decided to re-touch it, mostly for my enjoyment, to keep my software development edge, and to essentially make it more user friendly, give it better graphics and improve its performance. Since then, I have added many more features, improved its performance and will soon develop an Android and iOS version of it

Solitaire Chess has many features to make the game for the user as enjoyable as possible.

  • Completely Responsive Solitaire Chess components will adapt to any size (a minimum however is calculated).
  • Hint A button that uses Dijkstra's Algorithm to successfully find the next best move for the user.
  • Undo A button that a user can click to undo the latest move.
  • Reset A button that resets the game to the initial game.

  1. Move the chess pieces according to the movement rules (http://www.chessvariants.com/d.chess/chess.html).
    Each move MUST result in a caputred piece. Think ahead and plan your moves!
  2. If you are left with two or more pieces on the board you lose. Reset the challenge and try again.
  3. When there is only one piece left on the board, YOU WIN!
